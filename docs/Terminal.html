Terminal

Core Terminal class hosting a set of plugin and widget modules which together make up the interactive UI. For now limited to a single symbol, but the utilized data plugins are multi-symbol.

Verifies modules required by terminal are loaded, and emits/waits for the event:Terminal~terminalBooted event to complete. Calls Terminal.destroyWithError on failure to make the error visibile even after blessed teardown.
