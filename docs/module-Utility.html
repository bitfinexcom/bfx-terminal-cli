(static) measureDuration(asStringopt, coloropt, redThresholdopt) → {Array.<number>|Array.<string>}

Source:
  • util/measure_duration.js, line 36

Helper that returns a function which can be called multiple times to measure elapsed durations between calls.

Example
cosnt d = measureDuration(false)

  // long-running operations

  const first = d()
  const [firstDurationMS] = first
  const firstDurationString = first.toString()

  // second set of operations

  const second = d()
  const [secondDurationMS] = second
  const secondDurationString = second.toString()

  // etc
Parameters:
Name Type Attributes Default Description
asString boolean <optional>
 true

default true, if false returned value is the duration value in ms; otherwise a color formatted string
color function <optional>

color function for durations under red threshold
redThreshold number <optional>
 300

default 300ms, durations exceeding this value are colored red if asString is not false
Returns:

result - array with duration string (%dms) or value as single element, and modified toString() function returning the color-formatted string
